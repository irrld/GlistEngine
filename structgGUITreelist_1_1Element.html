◆ setIcon()

void gGUITreelist::Element::setIcon ( )
inline

Sets the icon attributes of all elements. This function uses when isicon value of the elements is true. Sets the icons according to their isparent attributes. If the element has got a sub vector, it's icon will be the 'Folder icon'. If it has not got a sub vector, it's icon will be the 'File icon'.
